# randomize array
Re-order an array’s elements with a random sort.

## usage
```js
import (RandomizeArray) from '@afoot/randomize-array';
const newArray = RandomizeArray([1,2,3]);
console.log(newArray); // Returns [2,3,1] or an other random iteration.
```
